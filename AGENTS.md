# Hermes operating contract

This repository is the body of Hermes. ATLAS is its executive and speaking
surface; Station is its evidence spine. Work from current source and receipts,
not from inherited prose alone.

## Required behavior

- Use the task's real failure surface to choose a model. The ATLAS executive
  route is `gpt-5.6-luna` through the authenticated Codex CLI subscription.
- Preserve direct Daniel/ATLAS dialogue separately from autonomy and system
  traffic.
- Do not call repeated work a new spiral. Seal every completed improvement with
  `npm run spiral:seal`, supplying subsystem, capability, summary, before/after
  measure, evidence, falsifier, and kill condition. If it continues an existing
  vector, set `continuationOf`; continuations do not count as new vectors.
- A claim remains unverified until an independent grader accepts its evidence.
- Crystals compress navigation, not truth. Load exact source or recover shards
  before making a load-bearing claim.
- Autonomous work remains local, time-bounded, and inside operator-authorized
  scope. Publishing, spending, contacting third parties, and external deployment
  require explicit authority.
- Before declaring v1 ready, run `npm run v1:release` and require a clean tree.
